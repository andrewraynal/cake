Overview

HappyCake is a Onepage Bussiness HTML Template designed in a clear & modern style. It's focused to introduce and show pastry cook's products. However, you can use template for other purposes. The HTMLs also very easy to customize. If you like my template, please rate it 5 stars. It’s very useful for me. Cheer!

Features

# Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. I used grid system an some features of bootstrap (v3.2.0). All editabe on main.css, included after bootstrap.css.

Learn more from http://getbootstrap.com/

# LessCss

LessCss is a dynamic css base on javascript. It's smart, fast code, clearly. LessCss was used when i develop but when finished, i compiled it (main.less) to css (main.css). It's better for performance. Need learn less css, go to lesscss.org.

To use LessCss, add this below or replace main.css.

<link rel="stylesheet/less" href="css/main.less">
<script src="js/less.min.js"></script>

# Icon Font (Awesomefont)

All icons were used in project is font. The complete set of 439 icons in Font Awesome 4.1.0. Very easy to user, edit, color it. Specialy it can be change size.

Bootstrap also included Glyphicons but not much icon like Font Awesome. To use, you can follow this way.

<i class="fa fa-nameicon"></i>

Learn more: http://fontawesome.io/icons/

# Animate Css

"Animate Css" have many beautiful css3 effects. Test it here http://daneden.github.io/animate.css/. To add animate on element you add animated class and name-of-effect class. Example:

<div id="element" class="animated name-of-effect">This is div</div>

PHP

This template is using one php file. A send mail function file named "sendmail.php". You can find it  in the attached template. Here are some config you need to change in order to work with your own account.

Sendmail.php: function file to send mail from contact section. Please change to your email address.

<?php
//This php file receives the post vars of the contact page and sends it to a email address

//Settings
$email = 'yourname@email.com';
$subject = '[NAME] has send you a mail from HappyCake';

//Get post variables
$name = $_POST['name'];
$from = $_POST['email'];
$message = $_POST['message'];

//Set the headers and send the mail
$subject = str_replace('[NAME]', $name, $subject);
$headers = 'From: '.$name.' <'.$from.'>';
//mail($email, $subject, $message, $headers,'-f '.$email);
mail($email, $subject, $message, $headers);
?>

Some Notices

Layout Boxed

Easily, you need add 'layout-boxed' on body. All content be wraped by a div has class "all".

<body class="layout-boxed">

Main Menu

Active menu when scroll to the section. Read more at http://getbootstrap.com/javascript/#scrollspy

<body data-spy="scroll" data-target=".main-nav" data-offset="70">

The main menu was cloned on Mt Sidebar. Mt Sidebar is a sidebar menu on small screen like tablet or mobile.

Fixed Footer.

You also fixed footer when you add class "fixed" into "footer-wrapper".

<div class="footer-wrapper fixed"></div>

If you fix footer, you should remove scrollspy on main menu because it work not extractly. The reason is footer-wrapper has attr is fixed, so scrollspy effect does'nt work like you want.
